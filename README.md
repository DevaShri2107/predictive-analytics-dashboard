# AI Predictive Analytics Dashboard

## Overview
A real-time business forecasting dashboard using Machine Learning and Streamlit.

The application predicts future sales trends using:
- Historical sales
- Advertising budgets
- Customer growth
- Profit analysis

---

## Features
- Interactive analytics dashboard
- Regression-based forecasting
- Data visualization
- KPI metrics
- Downloadable reports
- Region & category filters
- Future sales prediction

---

## Technologies Used
- Python
- Streamlit
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

---

## Run Project

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the app:

```bash
streamlit run app.py
```
